# 資工系網站技能索引

> **語言規範**：本文件使用**繁體中文**撰寫。

此目錄包含資工系網站相關的子技能。

## 使用方式

1. 先閱讀此 INDEX.md
2. 比對使用者請求與下方的觸發關鍵字
3. 若符合，則載入對應的 SKILL.md

---

## 可用子技能

### seminar-update

| 欄位 | 值 |
|------|-----|
| 路徑 | `./seminar-update/SKILL.md` |
| 說明 | 更新專題演講資料庫與網頁內容至新學期 |
| 觸發關鍵字 | 專題演講, seminar, 演講更新, 學期更新, 演講資料庫 |

**載入時機**：使用者需要建立新學期的專題演講 SQL table 或更新網頁內容。

---

### deploy

| 欄位 | 值 |
|------|-----|
| 路徑 | `./deploy/SKILL.md` |
| 說明 | 從 GitHub 拉取最新變更並自動上傳至 FTP |
| 觸發關鍵字 | 部署, deploy, 同步, sync, 上傳 FTP, 系網部署, 更新系網 |

**載入時機**：使用者已 push 到 GitHub，需要將變更同步到正式環境。
